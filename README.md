# Тестовый проект Cloudflare Pages

Это демо проект для деплоя на Cloudflare Pages.

## Что внутри:
- Статический HTML/CSS/JS сайт
- Пример Cloudflare Worker API
- Автоматический деплой из GitHub

## Как запустить:
1. Залить на GitHub
2. Подключить в Cloudflare Pages
3. Профит!

## Технологии:
- HTML5
- CSS3
- JavaScript
- Cloudflare Pages
- Cloudflare Workers (опционально)
